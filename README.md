# WeChatPro
WeChat Pro 2026 (Titan Edition) 是一款基于 Python 与 Windows 底层 API 构建的高性能微信自动化工具。

与传统的模拟点击脚本不同，Titan Edition 摒弃了不稳定的高层封装，直接与 Windows user32.dll 交互，并引入了独创的 “物理限速引擎” 与 “真人拟态算法”。旨在为社群运营、消息分发提供一个零封号风险、零资源残留的稳健解决方案。

核心代码完全运行于本地，不连接任何云端服务器，确保数据隐私绝对安全。
**⚠️ 郑重声明**：本项目基于稳定的 `UIAutomation` 和 `PyAutoGUI` 辅助驱动构建，不进行任何非法的内存注入或协议破解。工具仅供学习与提效交流，请勿用于黑灰产场景。

## ✨ 核心防御体系 (Security Core)

大多数自动化工具死于“特征码检测”和“行为分析”，WeChat Pro 2026 内置了三层防御盾牌，在操作系统的注视下变为“隐形人”：

1. **高级自动化引擎**：基于稳定且成熟的 `UIAutomation` 寻找与聚焦控件，配合精确的相对屏幕坐标计算，实现稳定兼容不同微信版本的自动化点击操控。
2. **UAC 提权伪装**：编译内置 `requireAdministrator` 清单注入。程序在启动那一刻即获得最高控制权，彻底规避 Windows UIPI 沙盒隔离拦截。
3. **独立运行环境 (Standalone 架构)**：摒弃了臃肿且容易触发杀软误报的 OneFile 模式，采用 Nuitka 完全编译为独立的文件夹系统。拥有原生级别的运行性能与完美的内存管理。

## 🚀 业务级特性 (Features)

* 🔤 **语义隐形引擎**：自动向发送内容中注入 24 种零宽混淆字符，确保每条发出的消息 Hash 值独一无二。
* 🖼️ **图片/视频隐形处理**：利用图像算法进行像素级微小扰动（肉眼不可见）；通过二进制写入技术在媒体文件尾部追加随机噪声数据，切断图文特征匹配。
* 🖱️ **真人拟态操作**：引入 **三阶贝塞尔曲线** 以随机控制点计算你的鼠标移动轨迹，配合微小随机手抖机制和疲劳拟态（Fatigue Breaks），行为模式与真实人类无异。
* 🗃️ **本地数据中枢**：采用本地 SQLite 建立数据舱，全面记录每一次触达行为，同时配套图形化的检视和清空工具进行可视化维护。
* 📝 **智能消息渲染模板**：通过 `{name}` `{date}` 以及 `{random:早上好|中午好}` 等关键字标签，使每一次群发都千变万化，实现自动化千人千面。
