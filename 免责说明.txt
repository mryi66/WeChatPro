📖 项目简介 (Introduction)

WeChat Pro 2026 (Titan Edition) 是一款基于 Python 与 Windows 底层 API 构建的高性能微信自动化工具。

与传统的模拟点击脚本不同，Titan Edition 摒弃了不稳定的高层封装，直接与 Windows user32.dll 交互，并引入了独创的 “物理限速引擎” 与 “真人拟态算法”。旨在为社群运营、消息分发提供一个零封号风险、零资源残留的稳健解决方案。

核心代码完全运行于本地，不连接任何云端服务器，确保数据隐私绝对安全。

🚀 核心特性 (Key Features)

1. 🛡️ 底层安全防御 (Deep Security)

直接 API 调用: 绕过上层封装，直接调用 ctypes.windll.user32 发送硬件级键鼠信号，在操作系统眼中与真实硬件操作无异。

内存驻留技术: 采用 OneFile 打包模式，所有依赖库与资源直接加载进内存数据段，运行时硬盘无临时文件残留，规避特征扫描。

2. 👻 智能分级隐形 (Adaptive Stealth 2.0)

语义混淆引擎: 自动识别发送频率。当检测到高频发送时，自动在文本中注入不可见的“零宽字符”(\u200b 等)，确保每一次发送内容的哈希值都独一无二。

媒体指纹重构: 内置图像隐形引擎，发送图片/视频前自动进行像素级微扰或二进制噪点注入，彻底防止文件 MD5 碰撞检测。

3. 🍃 真人拟态操作 (Human Mimicry)

非线性轨迹: 鼠标移动不再是机械的直线，而是基于贝塞尔曲线的平滑轨迹，并伴有随机微小的抖动（Jitter）。

疲劳模拟: 算法内置随机“疲劳值”，在连续操作一定次数后会自动触发随机时长的暂停，模拟真人的思考与休息间隙。

4. ⚡ 物理限速内核 (Physics-Based Rate Limiting)

摒弃了不稳定的 Turbo 极速模式，强制实施 0.05s 的物理 I/O 间隔，确保 Windows 消息队列永远不会阻塞，从根本上解决了程序卡死与崩溃问题。

Stealth: Pillow (Image Processing), Random Entropy Injection
